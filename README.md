# SeDeNet
TIFF Image Reading

To read 32-bit TIFF images, please install the ALBULA software provided in the 32bit_tiff_file reader.zip file. This software is required for proper visualization and handling of the TIFF images.

Environment Setup

Please configure the Python environment according to the specifications provided in environment_information.txt. Ensure that all required dependencies and versions are installed before running the code.

Model

The model directory contains pre-trained models that can be used directly.

If you wish to train the model yourself, please extract train_set.zip and use the extracted dataset for training.
